Sobre o SmartGrid

O SmartGrid é um sistema online para o monitoramento da rede doméstica de energia do usuário. É possível consultar o consumo através de verificações em tempo real e de registros armazenados.

Para que o sistema colete os dados de consumo, é necessário que o sistema receba as informações do cliente, através de um Arduino confeccionado especificamente para esta aplicação.
